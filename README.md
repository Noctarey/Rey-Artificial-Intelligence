# Analisis Komparatif LLM: Claude 4.6 vs Gemini 3.1 🚀

Repositori ini berisi makalah analisis mendalam mengenai perbandingan dua platform *Large Language Model* (LLM) terkini, yaitu **Claude 4.6** dari Anthropic dan **Gemini 3.1** dari Google DeepMind.

## 📝 Ringkasan Makalah

Makalah ini bertujuan untuk memberikan panduan objektif bagi pengembang dan organisasi dalam memilih model AI yang tepat. Kami membedah kedua model melalui empat dimensi utama:

1.  **Performa & Penalaran:** Claude 4.6 menunjukkan keunggulan pada tugas *coding* dan logika kompleks, sementara Gemini 3.1 sangat kuat dalam integrasi multimodal.
2.  **Efisiensi Operasional:** Analisis terhadap *Context Window* Gemini yang masif (hingga 2M token) dibandingkan dengan efisiensi biaya API Claude.
3.  **Arsitektur Teknis:** Pembahasan mengenai struktur *Transformer* dan *Mixture of Experts* (MoE) yang mendasari kedua model.
4.  **Dampak Lingkungan (Update):** Evaluasi jejak karbon (*carbon footprint*) dan pentingnya penggunaan "Green AI" melalui pemilihan model yang efisien secara energi.
